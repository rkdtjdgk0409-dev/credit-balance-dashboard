# 신용잔고·고객예탁금 대시보드 v8

노션 임베드와 GitHub Pages에서 사용하는 단순형 신용잔고 대시보드입니다.

## 화면 구성

1. 예탁금 대비 신용비율과 현재 위험 상태
2. 신용융자 잔고
3. 고객예탁금
4. 신용비율 구간별 지표 읽는 법

기존 추세 차트, 탭, Chart.js 의존성은 제거했습니다.

## 판정 기준

- 25% 미만: 매우 안정
- 25% 이상 35% 미만: 일반 수준
- 35% 이상 40% 미만: 주의 구간
- 40% 이상: 위험 구간

## 자동 갱신

실제 실행 파일은 .github/workflows/main.yml입니다.
평일 한국시간 오후 4시, 장 마감 후 한 번 실행합니다.
UTC 크론은 0 7 * * 1-5입니다.
