# 신용잔고·고객예탁금 대시보드 v7

GitHub Pages에서 실행되는 노션 임베드용 대시보드입니다.

## v7 변경점

- 최신 관측일 기준 **최근 6개월**만 저장·표시
- Hang on 동적 payload에서 전체 시계열을 우선 추출
- Hang on이 과거 payload를 노출하지 않을 때 금융투자협회 계열 증시자금 표로 과거 구간 보완
- 최신 행은 Hang on 화면의 기준일·신용융자·고객예탁금 값으로 검증 및 교체
- Chart.js의 `parsing: false` 오류 제거: 33% 데이터가 0~1% 축으로 보이던 문제 수정
- 첫 실행에서도 약 120~140영업일의 과거 데이터를 한 번에 수집

## 업로드

ZIP 안의 파일과 `.github` 폴더를 저장소 최상위에 덮어쓴 뒤 커밋합니다.

## 실행

Actions → 신용잔고 데이터 자동 갱신 → Run workflow

정상 실행 시 `data.json`의 `series`에 최소 20행 이상이 저장되고, 일반적으로 최근 6개월 영업일 데이터가 채워집니다.
